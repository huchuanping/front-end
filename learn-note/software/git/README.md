## 常用命令
* 从远程服务器克隆文件git clone
* 查看文件状态 git status
* 添加修改   git add -A 
* 提交改动到head git commit -m '注释'（这一步还没有把改动提交到仓库）
* 提交改动到远程仓库 git push
